# SMA Textiles Catalog Website

Premium, cinematic React + Tailwind + Framer Motion website for **SMA Textiles**.

## Features

- Animated loading intro
- Sticky/floating navigation with mobile menu
- Cinematic hero with parallax movement
- Brand statement section with signature card
- Premium category card grid
- Searchable + filterable product catalog
- Product quote CTA that auto-focuses inquiry section
- Immersive showcase storytelling section
- About + Why Choose Us sections
- Buyer trust badges
- Premium inquiry form + WhatsApp quick action
- SEO-ready title/description and semantic sections

## Tech Stack

- React (Vite)
- Tailwind CSS
- Framer Motion
- Lucide React

## Local Development

```bash
npm install
npm run dev
```

Open the local URL shown by Vite (default: `http://localhost:5173`).

## Production Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push this repository to GitHub.
2. Import project in Vercel.
3. Framework preset: **Vite**.
4. Build command: `npm run build`
5. Output directory: `dist`

## Deploy to Netlify

1. Connect repository in Netlify.
2. Build command: `npm run build`
3. Publish directory: `dist`

## Editable Data Files

- `src/data/products.js`
- `src/data/categories.js`

Update these files to add/remove catalog content without changing components.
